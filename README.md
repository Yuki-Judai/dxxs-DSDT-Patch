## 使用方法

如何在[MaciASL](http://sourceforge.net/projects/maciasl/)中加入这个源内的补丁: 

1. 运行 MaciASL
2. 在MaciASL的菜单栏上选择`配置`(`⌘,`)
3. 选择Sources
4. 点击`+`按钮
5. 输入源名字, 例如"派奇"
6. 输入如下URL: http://raw.github.com/Yuki-Judai/dxxs-DSDT-Patch/master
